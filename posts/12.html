# Linux/Ubuntu环境更新nodejs到最新版本

Please have a overlook of the following steps: (I don't use nvm or other tools)

sudo npm cache clean -f (force) clear you npm cache sudo npm install -g n in (opens new window)stall n (this might take a while) sudo n stable upgrade to the current stable version Note that sudo might prompt your password. But if you are working with WSL in Windows, you can get sudo by default.

Additional note regarding step 3: stable can be exchanged for latest, lts (long term support) or any specific version number such as 0.10.20.

If the version number doesn't show up when typing node -v, you might have to reboot.

When your envrionment still show the old version after node -v, you can type hash -r and try again.

These instructions are found here as well: davidwalsh.name/upgrade-nodejs (opens new window) More info about the n package found here: npmjs.com/package/n (opens new window) More info about Node.js' release schedule: github.com/nodejs/Release (opens new window)
